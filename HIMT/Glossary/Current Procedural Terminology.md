# Current Procedural Terminology (CPT)
*A code set copyrighted and maintained by the [American Medical Association](american-medical-association.md) (AMA). Current Procedural Terminology is a five-digit code used to describe medical, surgical, radiology, laboratory, anesthesiology, and evaluation/management services of physicians, hospitals, and other healthcare providers.*
